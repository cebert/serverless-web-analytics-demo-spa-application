Welcome to the Serverless Website Analytics Demo AWS Community Day 2024, Columbus, Ohio

This is a simple Angular SPA application that uses the Serverless Website Analytics SDK Client to ingest logs into Serverless Website Analytics.

Using the SDK Client is not strictly necessary. You can ingest data into Serverless Website Analytics one of the following ways:

  1. Import Script
    • Add an import script to the site's HTML load ingestion code.
    • When the attribute tracking option is enabled, only buttons and links with the swa-event attribute are tracked.
  2. SDK Client
    • Import SDK Client package for more control over event publication. This approach works better for SPA applications with complex routing.

What is Serverless Website Analytics?

Serverless website analytics is an open source project that can be used to collect and analyzing website visitor data without managing any infrastructure. It leverages serverless technologies such as AWS Lambda, API Gateway, and DynamoDB to provide scalable, cost-effective analytics solutions. You may want to consider including it in your next project or personal website in order to collect data on visitors to your site.
